# kogakure.de

This the source of my [Ninja and Ninjutsu book][kogakure] build with [Eleventy], [PostCSS], and [Gulp].

## Installation

```sh
npm install
```

## Running the Website

```sh
npm start
```

For more tasks have a look into the `package.json`.
